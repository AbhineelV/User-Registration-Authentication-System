# User Registration & Login Authentication System
## 📌 Project Overview

This project is a web-based user authentication system developed using ASP.NET Web Forms and SQL Server. It allows users to register, store their details in a database, and securely log in using their credentials.

The system demonstrates core backend development concepts such as database connectivity, session management, and form handling.

## 🚀 Features
- User Registration (Name, Mobile, Email, Password)
- User Login Authentication
- Session Management for logged-in users
- Database connectivity using ADO.NET
- Parameterized queries to prevent SQL Injection

## 🛠️ Technologies Used
- ASP.NET Web Forms (C#)
- SQL Server
- ADO.NET (SqlConnection, SqlCommand, SqlDataReader)
- Visual Studio
